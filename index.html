Description

ALCHEM (Architecture Lab for Creative High-performance Energy-efficient Machines) at Purdue University focuses on computer architectures and systems for important applications such as graph processing, machine learning, with emerging technologies such as ReRAM, Processing-In-Memory and Non-Volatile Memory, etc. Our research takes a vertical approach, which uniquely tackles the challenges by focusing on the interactions across multiple system stacks including applications, programming model, runtime/compiler, architecture and emerging technology. This approach fully exploits the opportunities that are not sufficient, or even possible if only one system layer is considered. If interested, please find more information from the summary of our research projects.

Research Projects

  • Graph Processing System and Architecture
  • Machine Learning Acceleration
  • Storage and Cloud Systems
  • GPU Architecture
  • Parallelism and Concurrency
  • Quantum Computing
  • Hardware Security
